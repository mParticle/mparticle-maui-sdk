# mParticle .NET MAUI SDK

## Overview

The mParticle .NET MAUI SDK contains bindings for the mParticle native iOS and Android mobile SDKs.
It provides a common API interface for ease of integration with iOS and Android targets.

## Project Structure

The project utilizes [Maui Native Library Interop](https://github.com/CommunityToolkit/Maui.NativeLibraryInterop)
to create the bindings. These bindings are used by the mParticle SDK wrapper APIs to expose the native library functionality.

## Supported Targets

- .NET iOS
- .NET Android
